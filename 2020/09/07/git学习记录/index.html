（2）小结

Git 鼓励大量使用分支：
查看分支：git branch
创建分支：git branch 分支名
切换分支：git checkout 分支名
创建+切换分支：git checkout -b
合并某分支到当前分支：git merge
删除分支：git branch -d
当 Git 无法自动合并分支时，就必须首先解决冲突。解决冲突后，再提交，合并完成。用 git log –graph 命令可以看到分支合并图。
